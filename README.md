# HR Diagram Analysis

## 📌 Description
This project analyzes observational stellar data to calculate luminosity and temperature and generate a Hertzsprung-Russell (HR) diagram.

## ⚙️ Features
- Data loading from .txt and Excel files
- Data processing using pandas
- Visualization of stellar properties
- HR diagram generation

## 🛠️ Technologies
- Python
- Pandas
- Matplotlib

## 🎯 Purpose
Apply data analysis and visualization techniques to scientific datasets.
